#Fullstack Wolt-like application

Architecture:
- Backend: Spring Boot + JPA + MySql (XAMPP)
- Desktop: Hibernate(ORM) + JPA + MySql (XAMPP)
- Android: RESTful API from Backend
- Encryption: BCrypt for Password hashing

Features:
- User management
- Order management
- Menu/Cuisine management
- Chat system
- Order pickup/delivery system
- Review management (WIP probably will never finish lol)
